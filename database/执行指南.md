## 🔧 常见问题解决

### 问题1：MySQL命令未找到
**解决方案**：
1. 将MySQL的bin目录添加到系统PATH环境变量
   - 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
   - 在系统变量中找到Path，添加：`C:\Program Files\MySQL\MySQL Server 8.0\bin`
2. 或者使用MySQL完整路径执行命令

### 问题2：无法连接MySQL服务器
**解决方案**：
1. 检查MySQL服务是否启动
   ```powershell
   # 查看服务状态
   Get-Service MySQL*
   
   # 启动服务
   net start MySQL80
   ```
2. 检查防火墙设置
3. 确认用户名和密码正确

### 问题3：执行SQL脚本时报错
**解决方案**：
1. 检查SQL文件路径是否正确（使用正斜杠/或双反斜杠\\）
2. 确保MySQL用户有创建数据库的权限
3. 查看具体错误信息，根据错误提示解决

### 问题4：字符集问题
**解决方案**：
- 确保MySQL配置文件(my.ini)中设置了正确的字符集：
```ini
[mysqld]
character-set-server=utf8mb4
collation-server=utf8mb4_unicode_ci

[client]
default-character-set=utf8mb4
```
