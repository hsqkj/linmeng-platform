# 邻盟营销助手 - 完整使用指南

## 📋 项目概述

**邻盟营销助手**是一个专为社区商户打造的智能营销推广平台，帮助商家快速发布营销信息、触达周边居民、提升经营效率。系统采用"极简操作+智能优化+多渠道发布"的设计理念，降低商户营销门槛，实现从内容制作到用户转化的完整闭环。

### 核心价值

- **降低门槛**：模板化发布，3步完成营销内容制作
- **智能优化**：AI自动优化内容，提升营销效果和合规性
- **多渠道发布**：一键发布到朋友圈、抖音、小红书等5大渠道
- **精准触达**：基于LBS定位，精准触达周边3公里居民
- **数据驱动**：实时统计浏览、咨询、订单数据，优化营销策略

## 🎯 目标用户

### 商家端
- 社区便利店、餐饮店、生鲜超市、家政服务、维修服务等
- 特点：认知水平较低、操作能力弱、对复杂功能接受度低
- 需求：极简操作流程、快速发布、效果可见

### 用户端
- 社区居民
- 特点：追求便利、关注优惠、喜欢分享
- 需求：便捷查看、快速咨询、轻松下单

## 🏗️ 技术架构

### 前端技术栈
- **HTML5 + CSS3 + JavaScript**：原生技术栈，轻量高效
- **Tailwind CSS**：实用优先的CSS框架，快速构建美观界面
- **Lucide Icons**：轻量级图标库，提供丰富的图标资源
- **LocalStorage**：浏览器本地存储，实现数据持久化

### 后端技术栈（规划中）
- **Node.js + Express**：高性能Web服务框架
- **MySQL**：关系型数据库，存储核心业务数据
- **Redis**：缓存系统，提升数据访问速度
- **JWT**：身份认证，保障系统安全

### AI服务（规划中）
- **OpenAI API / 百度文心一言**：内容智能优化
- **讯飞语音识别API**：语音转文字
- **百度OCR API**：图片文字识别

## 🚀 快速开始

### 方法1：使用启动脚本（推荐）

**Windows用户：**
```bash
双击运行 start.bat 或 start-simple.bat
```

**Linux/Mac用户：**
```bash
chmod +x start.sh
./start.sh
```

### 方法2：手动启动

**使用Python：**
```bash
python -m http.server 8000
```

**使用Node.js：**
```bash
npx http-server -p 8000
```

### 方法3：直接打开（离线版本）

如果遇到网络问题或CDN加载失败，可以使用离线版本：

- **商家端离线版**：直接打开 `merchant/post-message/index-offline.html`
- **用户端离线版**：直接打开 `resident/view-messages/index-offline.html`

离线版本不依赖外部CDN，可以在没有网络的情况下使用。

### 访问应用

启动成功后，在浏览器中访问：

- **商家端（在线版）**：http://localhost:8000/merchant/post-message/index.html
- **用户端（在线版）**：http://localhost:8000/resident/view-messages/index.html
- **商家端（离线版）**：file:///path/to/merchant/post-message/index-offline.html
- **用户端（离线版）**：file:///path/to/resident/view-messages/index-offline.html

## 📖 功能说明

### 商家端功能

#### 1. 发布营销信息（4步流程）

**步骤1：选择模板**
- 10种营销模板：新店开业、限时折扣、套餐优惠、活动通知、上门服务、拼团集赞、节日特惠、会员专享、尾货清仓、自定义
- 模板分类筛选：全部、促销活动、新品上市、服务预约、会员专享
- 点击模板自动填充标题和内容

**步骤2：编辑内容**
- 营销标题输入
- 内容类型选择：图文/短视频
- 营销内容编辑：
  - 语音输入：支持普通话语音识别（使用浏览器Web Speech API）
  - 图片识别：OCR文字识别（预留接口）
  - 文本编辑：支持多行文本输入
- 图片/视频上传：支持多图上传，最多9张
- 标签选择：预设标签+自定义标签
- 附加功能：允许留言、允许分享、服务预约、拼团功能

**步骤3：AI智能优化**
- 优化维度：
  - 语句通顺度：优化语言表达
  - 营销吸引力：增强营销话术
  - 社区场景适配：适配社区语境
  - 广告法合规：过滤违规词汇
- 优化结果对比：原内容 vs 优化后内容
- 一键替换/保留原内容

**步骤4：渠道发布**
- 发布渠道选择：
  - 朋友圈：微信朋友圈发布
  - 抖音：短视频平台发布
  - 小红书：种草社区发布
  - 益邻邻小程序：小程序内发布
  - 好社区惠民社群：社区群发布
- 目标人群设置：
  - 周边范围：1.0km/1.5km/2.0km/3.0km
  - 人群类型：所有居民/会员用户/新用户/老用户
- 发布时间设置：
  - 立即发布
  - 定时发布

#### 2. 自动保存功能
- 每3秒自动保存草稿
- 草稿保留7天
- 支持恢复上次编辑内容

#### 3. 查看已发布消息
- 消息列表展示
- 消息详情查看
- 消息状态管理

#### 4. 数据统计
- 浏览量、点赞数、评论数、分享数、订单数统计
- 按日/周/月筛选
- 数据可视化展示（图表）

### 用户端功能

#### 1. 首页浏览
- 活动列表展示（按距离优先排序）
- 搜索功能：搜索商家、活动、优惠
- 分类筛选：全部、新店开业、限时折扣、套餐优惠、上门服务、拼团集赞
- 社区选择：切换不同社区
- 收藏功能：收藏感兴趣的活动

#### 2. 活动详情
- 完整活动信息展示
- 商家信息展示
- 活动内容展示（图文/视频）
- 互动数据：浏览量、点赞数、评论数、分享数
- 用户评论展示

#### 3. 咨询商家
- 无需注册即可咨询
- 微信快捷登录
- 发送文字咨询
- 商家24小时内回复

#### 4. 立即下单
- 简化下单表单：联系人、联系电话、下单内容、数量
- 快速提交订单
- 订单状态跟踪

#### 5. 收藏管理
- 收藏活动列表
- 取消收藏
- 快速访问收藏活动

#### 6. 订单管理
- 订单列表展示
- 订单详情查看
- 订单状态跟踪

## 📱 使用指南

### 商家使用流程

1. **注册登录**
   - 访问商家端页面
   - 填写商家信息（名称、联系人、手机号、地址等）
   - 提交注册

2. **发布营销信息**
   - 选择合适的营销模板
   - 编辑活动内容（支持语音输入、图片上传）
   - 使用AI优化内容
   - 选择发布渠道和目标人群
   - 设置发布时间
   - 一键发布

3. **查看数据**
   - 访问数据统计页面
   - 查看浏览量、咨询数、订单数
   - 分析营销效果
   - 优化营销策略

### 用户使用流程

1. **浏览活动**
   - 访问用户端页面
   - 选择所在社区
   - 浏览周边营销活动
   - 使用搜索和筛选功能

2. **查看详情**
   - 点击活动卡片查看详情
   - 了解活动内容、商家信息
   - 查看用户评论

3. **互动下单**
   - 咨询商家：发送文字咨询
   - 立即下单：填写简单表单提交订单
   - 收藏活动：收藏感兴趣的活动

## 🎨 界面设计

### 设计原则

- **极简布局**：顶部标题栏+中间内容区+底部操作栏
- **视觉层级**：标题18px/黑色、内容14px/深灰、提示12px/浅灰
- **操作引导**：关键步骤有文字+图标引导
- **适配性**：支持375px-414px主流手机宽度

### 色彩方案

- **主色调**：橙色 (#f97316) - 活力、温暖
- **辅助色**：
  - 绿色 (#10b981) - 成功、确认
  - 蓝色 (#3b82f6) - 信息、链接
  - 紫色 (#8b5cf6) - 会员、专属
  - 红色 (#ef4444) - 紧急、清仓
- **中性色**：
  - 深灰 (#1f2937) - 标题
  - 中灰 (#4b5563) - 正文
  - 浅灰 (#6b7280) - 提示
  - 超浅灰 (#f3f4f6) - 背景

## 🔒 安全策略

### 数据安全
- 敏感信息（手机号、微信号）AES加密存储
- 查询结果脱敏展示
- 定期数据备份

### 内容安全
- AI过滤违规词
- 图片内容审核（预留）
- 敏感信息脱敏

### 接口安全（规划中）
- JWT身份认证
- 接口签名验证
- 请求频率限制
- SQL注入防护
- XSS攻击防护
