# 邻盟营销助手 - 快速使用指南

## 🎯 系统简介

邻盟营销助手是一个专为社区商户打造的智能营销推广平台，帮助商家快速发布营销信息、触达周边居民、提升经营效率。

## 🚀 快速开始

### 启动方式

#### 方式1：使用启动脚本（推荐）

**Windows用户：**
- 双击运行 `start.bat` 或 `start-simple.bat`

**Linux/Mac用户：**
```bash
chmod +x start.sh
./start.sh
```

#### 方式2：手动启动

```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx http-server -p 8000
```

#### 方式3：直接打开离线版本

如果遇到网络问题或CDN加载失败，可以直接打开离线版本：

- **商家端**：`merchant/post-message/index-offline.html`
- **用户端**：`resident/view-messages/index-offline.html`

### 访问地址

启动成功后，在浏览器中访问：

- **商家端**：http://localhost:8000/merchant/post-message/index.html
- **用户端**：http://localhost:8000/resident/view-messages/index.html

## 📱 商家端使用指南

### 第一步：选择营销模板

1. 进入商家端页面
2. 从10种营销模板中选择一个：
   - 新店开业
   - 限时折扣
   - 套餐优惠
   - 活动通知
   - 上门服务
   - 拼团集赞
   - 节日特惠
   - 会员专享
   - 尾货清仓
   - 自定义
3. 点击模板，系统会自动填充标题和内容

### 第二步：编辑营销内容

1. **填写营销标题**：输入吸引人的标题
2. **选择内容类型**：图文或短视频
3. **编辑营销内容**：
   - 使用语音输入：点击"语音输入"按钮，说出内容
   - 使用图片识别：点击"图片识别"按钮，上传图片识别文字
   - 直接输入：在文本框中输入内容
4. **上传图片/视频**：点击上传区域，选择文件（最多9张）
5. **选择标签**：点击预设标签或输入自定义标签
6. **设置附加功能**：
   - 允许留言
   - 允许分享
   - 服务预约
   - 拼团功能

### 第三步：AI智能优化

1. 选择优化维度：
   - 语句通顺度
   - 营销吸引力
   - 社区场景适配
   - 广告法合规
2. 点击"开始AI优化"按钮
3. 查看优化结果对比
4. 选择"使用优化内容"或"保留原内容"

### 第四步：渠道发布

1. **选择发布渠道**（可多选）：
   - 朋友圈
   - 抖音
   - 小红书
   - 益邻邻小程序
   - 好社区惠民社群
2. **设置目标人群**：
   - 周边范围：1.0km/1.5km/2.0km/3.0km
   - 人群类型：所有居民/会员用户/新用户/老用户
3. **设置发布时间**：
   - 立即发布
   - 定时发布
4. 点击"发布营销信息"按钮

### 自动保存功能

- 系统每3秒自动保存草稿
- 草稿保留7天
- 刷新页面后会自动恢复上次编辑内容

## 👥 用户端使用指南

### 浏览营销活动

1. 进入用户端页面
2. **搜索功能**：在搜索框输入关键词搜索商家、活动、优惠
3. **分类筛选**：点击分类标签筛选活动：
   - 全部
   - 新店开业
   - 限时折扣
   - 套餐优惠
   - 上门服务
   - 拼团集赞
4. **切换社区**：点击定位图标，选择所在社区
5. **查看活动列表**：浏览周边营销活动

### 查看活动详情

1. 点击活动卡片查看详情
2. 查看完整活动信息、商家信息
3. 查看互动数据（浏览量、点赞数、评论数、分享数）
4. 查看用户评论

### 咨询商家

1. 在活动详情页点击"咨询商家"按钮
2. 输入咨询内容
3. 点击"发送咨询"
4. 商家将在24小时内回复

### 立即下单

1. 在活动详情页点击"立即下单"按钮
2. 填写下单信息：
   - 联系人
   - 联系电话
   - 下单内容
   - 数量
3. 点击"提交订单"
4. 商家将尽快联系您

### 收藏活动

1. 在活动详情页点击收藏图标
2. 点击底部导航栏的"收藏"查看收藏的活动
3. 可以取消收藏

### 查看订单

1. 点击底部导航栏的"订单"
2. 查看所有订单列表
3. 点击订单查看详情
4. 跟踪订单状态

## 💡 使用技巧

### 商家端技巧

1. **选择合适的模板**：根据活动类型选择最匹配的模板
2. **充分利用AI优化**：让AI帮您提升内容质量
3. **多渠道发布**：选择多个发布渠道，扩大覆盖面
4. **设置合理的目标人群**：根据活动特点选择合适的人群范围
5. **定时发布**：选择用户活跃时间发布，提升效果
6. **查看数据统计**：定期查看浏览量、咨询数、订单数，优化营销策略

### 用户端技巧

1. **使用搜索功能**：快速找到感兴趣的活动
2. **收藏活动**：收藏感兴趣的活动，方便后续查看
3. **及时咨询**：有疑问及时咨询商家，获取更多信息
4. **分享活动**：分享给朋友，一起参与活动
5. **关注商家**：关注喜欢的商家，及时获取优惠信息

## ⚠️ 注意事项

### 商家端注意事项

1. **内容合规**：确保营销内容符合广告法规定
2. **信息准确**：活动时间、优惠信息等要准确无误
3. **及时回复**：及时回复用户咨询，提升用户体验
4. **定期更新**：定期发布新的营销活动，保持活跃度
5. **数据分析**：关注数据统计，优化营销策略

### 用户端注意事项

1. **核实信息**：下单前核实活动信息，避免误解
2. **保护隐私**：不要泄露个人敏感信息
3. **理性消费**：根据实际需求下单，避免冲动消费
4. **及时沟通**：有问题及时与商家沟通
5. **评价反馈**：活动结束后给予评价，帮助其他用户

## 🆘 常见问题

### 商家端常见问题

**Q: 如何修改已发布的活动？**
A: 目前版本暂不支持修改已发布活动，建议删除后重新发布。

**Q: AI优化功能收费吗？**
A: 当前版本AI优化功能免费使用。

**Q: 支持哪些图片格式？**
A: 支持PNG、JPG、WEBP格式，最多上传9张图片。

**Q: 如何查看数据统计？**
A: 点击顶部"数据统计"按钮查看详细数据。

**Q: 草稿会自动删除吗？**
A: 草稿会保留7天，7天后自动删除。

### 用户端常见问题

**Q: 如何切换社区？**
A: 点击顶部定位图标，选择所在社区。

**Q: 如何联系商家？**
A: 在活动详情页点击"咨询商家"按钮。

**Q: 订单提交后可以取消吗？**
A: 订单提交后如需取消，请直接联系商家。

**Q: 如何查看我的订单？**
A: 点击底部导航栏的"订单"查看。

**Q: 收藏的活动会丢失吗？**
A: 收藏活动保存在本地，清除浏览器数据会丢失。
