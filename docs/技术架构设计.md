## 2. 技术栈选型

### 2.1 前端技术栈
```
小程序端：
- 微信小程序原生框架
- WXML + WXSS + JavaScript
- 微信云开发（可选）

Web管理端：
- Vue 3 + Vite
- Element Plus / Ant Design Vue
- Axios
- ECharts / Chart.js
```

### 2.2 后端技术栈
```
主服务：
- Node.js + Express / Koa
- TypeScript
- JWT认证
- Multer文件上传

AI服务：
- Python + FastAPI
- OpenAI API / 百度文心一言
- 讯飞语音识别API
- 百度OCR API
```

### 2.3 数据库技术栈
```
关系型数据库：
- MySQL 8.0
- Sequelize ORM

缓存：
- Redis 6.0
- Redisson

文档数据库：
- MongoDB 5.0
- Mongoose

对象存储：
- 阿里云OSS / 腾讯云COS
```

### 2.4 部署架构
```
- Docker + Docker Compose
- Nginx反向代理
- PM2进程管理
- CI/CD：GitHub Actions / GitLab CI
```

## 3. 数据库设计

### 3.1 核心数据表

#### 3.1.1 用户相关表

**商户账号表 (merchant_accounts)**
```sql
CREATE TABLE merchant_accounts (
  id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '商户ID',
  openid VARCHAR(100) UNIQUE COMMENT '微信OpenID',
  phone VARCHAR(20) UNIQUE COMMENT '手机号',
  password VARCHAR(255) COMMENT '密码（加密）',
  status TINYINT DEFAULT 1 COMMENT '状态：1正常 2禁用',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_openid (openid),
  INDEX idx_phone (phone)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='商户账号表';
```

**商户信息表 (merchant_info)**
```sql
CREATE TABLE merchant_info (
  id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '商户信息ID',
  merchant_id BIGINT NOT NULL COMMENT '商户ID',
  name VARCHAR(100) NOT NULL COMMENT '商户名称',
  contact_name VARCHAR(50) COMMENT '联系人姓名',
  phone VARCHAR(20) COMMENT '联系电话',
  address VARCHAR(255) COMMENT '详细地址',
  latitude DECIMAL(10,7) COMMENT '纬度',
  longitude DECIMAL(10,7) COMMENT '经度',
  community_id INT COMMENT '所属社区ID',
  industry_id INT COMMENT '行业ID',
  business_license VARCHAR(255) COMMENT '营业执照URL',
  avatar VARCHAR(255) COMMENT '商户头像',
  description TEXT COMMENT '商户简介',
  business_hours VARCHAR(100) COMMENT '营业时间',
  status TINYINT DEFAULT 1 COMMENT '状态：1正常 2审核中 3禁用',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_merchant_id (merchant_id),
  INDEX idx_community_id (community_id),
  INDEX idx_industry_id (industry_id),
  INDEX idx_location (latitude, longitude)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='商户信息表';
```

**用户表 (users)**
```sql
CREATE TABLE users (
  id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '用户ID',
  openid VARCHAR(100) UNIQUE COMMENT '微信OpenID',
  nickname VARCHAR(50) COMMENT '昵称',
  avatar VARCHAR(255) COMMENT '头像',
  phone VARCHAR(20) COMMENT '手机号',
  gender TINYINT COMMENT '性别：0未知 1男 2女',
  community_id INT COMMENT '所属社区ID',
  member_level TINYINT DEFAULT 0 COMMENT '会员等级：0普通 1银卡 2金卡 3钻石',
  points INT DEFAULT 0 COMMENT '积分',
  status TINYINT DEFAULT 1 COMMENT '状态：1正常 2禁用',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_openid (openid),
  INDEX idx_community_id (community_id),
  INDEX idx_member_level (member_level)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户表';
```

#### 3.1.2 营销活动相关表

**营销活动表 (marketing_activities)**
```sql
CREATE TABLE marketing_activities (
  id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '活动ID',
  merchant_id BIGINT NOT NULL COMMENT '商户ID',
  title VARCHAR(200) NOT NULL COMMENT '活动标题',
  content TEXT NOT NULL COMMENT '活动内容',
  template_type VARCHAR(50) COMMENT '模板类型',
  cover_image VARCHAR(255) COMMENT '封面图',
  images JSON COMMENT '图片列表',
  video_url VARCHAR(255) COMMENT '视频URL',
  tags JSON COMMENT '标签列表',
  start_time DATETIME COMMENT '开始时间',
  end_time DATETIME COMMENT '结束时间',
  publish_time DATETIME COMMENT '发布时间',
  status TINYINT DEFAULT 0 COMMENT '状态：0草稿 1已发布 2已下架',
  views INT DEFAULT 0 COMMENT '浏览量',
  likes INT DEFAULT 0 COMMENT '点赞数',
  shares INT DEFAULT 0 COMMENT '分享数',
  comments INT DEFAULT 0 COMMENT '评论数',
  orders INT DEFAULT 0 COMMENT '订单数',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_merchant_id (merchant_id),
  INDEX idx_status (status),
  INDEX idx_publish_time (publish_time),
  INDEX idx_template_type (template_type),
  FULLTEXT idx_content (title, content)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='营销活动表';
```

**多渠道内容表 (channel_contents)**

CREATE TABLE channel_contents (

  INDEX idx_activity_id (activity_id),
  INDEX idx_channel_type (channel_type),
