# 邻盟营销助手需求文档

## 1. 项目概述

### 1.1 项目背景
邻盟营销助手是一个面向社区商户的智能营销推广平台，旨在帮助社区商户快速创建、优化和发布营销内容到多个社交平台，提升商户的营销效率和品牌影响力。

### 1.2 项目目标
- 降低社区商户营销内容制作门槛
- 提供AI智能优化功能，提升内容质量
- 支持多渠道一键发布，扩大营销覆盖面
- 构建社区商户与居民的连接平台

### 1.3 目标用户
- **主要用户**：社区商户（餐饮、零售、服务业等）
- **次要用户**：社区居民（浏览营销信息、收藏、下单）

## 2. 功能需求

### 2.1 用户管理模块

#### 2.1.1 商家注册
- **功能描述**：新商户注册账号并完善店铺信息
- **需求详情**：
  - 支持手机号注册
  - 支持微信一键登录
  - 新用户自动创建基础账号
  - 引导完善店铺信息（店铺名称、行业、地址、营业时间、社区、logo等）
  - 支持上传店铺logo和门头照片
  - 支持地图定位店铺位置
  - 支持选择所在社区
  - 表单验证和错误提示

#### 2.1.2 商家登录
- **功能描述**：已注册商户登录系统
- **需求详情**：
  - 支持手机号登录
  - 支持微信一键登录
  - 自动登录状态保持
  - 登录失败提示
  - 新用户自动注册流程

### 2.2 营销内容创建模块

#### 2.2.1 模板选择
- **功能描述**：提供多种营销模板供商户选择
- **需求详情**：
  - 支持多种模板类型：
    - 新品上市
    - 限时优惠
    - 满减活动
    - 注册有礼
    - 转发有奖
    - 招募代理
    - 招工用工
    - 招商加盟
    - 节日促销
    - 会员活动
  - 模板图标和预览
  - 点击选择模板
  - 模板切换功能

#### 2.2.2 内容编辑
- **功能描述**：基于模板创建和编辑营销内容
- **需求详情**：
  - 标题输入（字数限制和实时显示）
  - 详情内容编辑（支持多行文本）
  - 图片上传（支持多张图片，自动调整尺寸）
  - 视频上传（可选）
  - 字数统计和限制提示
  - 实时预览功能
  - 草稿保存功能

#### 2.2.3 AI智能优化
- **功能描述**：使用AI技术优化营销内容
- **需求详情**：
  - 四个优化维度：
    - 文案优化
    - 排版优化
    - 表情符号优化
    - 关键词优化
  - 原文与优化内容对比显示
  - 支持编辑优化后的内容
  - 支持重新优化
  - 优化结果预览

### 2.3 多渠道发布模块

#### 2.3.1 渠道选择
- **功能描述**：选择营销内容发布的目标平台
- **需求详情**：
  - 支持的发布渠道：
    - 小红书
    - 微信视频号
    - 抖音
    - 微信朋友圈
  - 点击渠道显示该渠道专属内容
  - 渠道内容自动切换
  - 支持多渠道选择

#### 2.3.2 渠道内容定制
- **功能描述**：为不同渠道生成适配的内容
- **需求详情**：
  - AI自动生成渠道专属文案
  - 支持手动编辑渠道内容
  - 渠道内容预览
  - 字数限制和提示

#### 2.3.3 一键发布
- **功能描述**：同时发布到多个选定渠道
- **需求详情**：
  - 检查必填项
  - 验证渠道选择
  - 发布进度显示
  - 发布结果反馈
  - 发布历史记录

### 2.4 商家信息管理模块

#### 2.4.1 店铺信息完善
- **功能描述**：管理和更新店铺基本信息
- **需求详情**：
  - 店铺名称编辑
  - 行业分类选择
  - 详细地址输入
  - 营业时间设置
  - 社区选择
  - 地图定位
  - Logo上传和更换
  - 门头照片上传
  - 联系方式管理

#### 2.4.2 数据统计
- **功能描述**：查看营销效果数据
- **需求详情**：
  - 发布次数统计
  - 浏览量统计
  - 收藏量统计
  - 订单量统计
  - 数据图表展示
  - 时间范围筛选

### 2.5 居民端功能模块

#### 2.5.1 营销信息浏览
- **功能描述**：居民浏览社区商户的营销信息
- **需求详情**：
  - 信息列表展示
  - 按商户分类
  - 按距离排序
  - 搜索功能
  - 详情页面查看

#### 2.5.2 商家详情
- **功能描述**：查看商家详细信息
- **需求详情**：
  - 商家logo和名称
  - 营业时间
  - 地址和地图
  - 联系方式
  - 商家发布的所有营销信息

#### 2.5.3 收藏功能
- **功能描述**：收藏感兴趣的营销信息
- **需求详情**：
  - 一键收藏
  - 收藏列表查看
  - 取消收藏

#### 2.5.4 订单管理
- **功能描述**：管理基于营销信息的订单
- **需求详情**：
  - 订单列表
  - 订单详情
  - 订单状态跟踪
  - 订单操作（确认、取消等）

## 3. 技术需求

### 3.1 前端技术
- **开发框架**：HTML5 + CSS3 + JavaScript
- **UI框架**：Tailwind CSS
- **响应式设计**：移动端优先，适配各种屏幕尺寸
- **数据存储**：LocalStorage
- **地图服务**：HTML5 Geolocation API

### 3.2 后端技术
- **Web服务器**：Node.js + http-server
- **API接口**：RESTful API
- **数据存储**：文件系统（开发阶段）

### 3.3 性能要求
- 页面加载时间 < 3秒
- 图片上传响应时间 < 5秒
- AI优化响应时间 < 10秒
- 支持并发用户数 > 100

### 3.4 兼容性要求
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 支持移动端浏览器
- 支持微信内置浏览器
