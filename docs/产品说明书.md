# 邻盟营销助手产品说明书

## 产品简介

邻盟营销助手是一款专为社区商户打造的智能营销推广平台，帮助商户快速创建、优化和发布营销内容到多个社交平台。通过AI智能优化技术，商户可以轻松制作专业的营销文案，一键发布到小红书、微信视频号、抖音等平台，大幅提升营销效率和品牌影响力。

## 产品特色

### 🤖 AI智能优化
- 四大优化维度：文案优化、排版优化、表情符号优化、关键词优化
- 原文与优化内容对比，支持二次编辑
- 针对不同平台自动生成适配内容

### 📱 多渠道发布
- 支持小红书、微信视频号、抖音、微信朋友圈
- 一键发布到多个平台
- 渠道内容自动切换和预览

### 🎨 丰富模板
- 新品上市、限时优惠、满减活动等多种模板
- 一键套用，快速创建专业内容
- 支持自定义编辑

### 📊 数据统计
- 发布次数、浏览量、收藏量、订单量统计
- 可视化数据图表
- 帮助商户了解营销效果

## 商家端使用指南

### 1. 注册登录

#### 1.1 新用户注册
1. 访问商家登录页面
2. 输入手机号或使用微信一键登录
3. 系统自动创建账号
4. 按照提示完善店铺信息

#### 1.2 完善店铺信息
1. **基本信息**
   - 店铺名称：输入您的店铺名称
   - 行业分类：选择所属行业（餐饮、零售、服务等）
   - 联系电话：输入店铺联系电话

2. **位置信息**
   - 点击地图定位按钮，获取当前位置
   - 输入详细地址
   - 选择所在社区

3. **营业信息**
   - 设置营业时间（如：09:00-21:00）
   - 上传店铺logo（建议尺寸：200x200像素）
   - 上传门头照片（建议尺寸：800x400像素）

4. **提交保存**
   - 检查信息无误后点击提交
   - 系统提示保存成功

#### 1.3 老用户登录
1. 访问商家登录页面
2. 输入已注册的手机号
3. 点击登录按钮
4. 进入营销内容创建页面

### 2. 创建营销内容

#### 2.1 选择模板
1. 进入营销内容创建页面
2. 浏览可用的营销模板
3. 点击选择合适的模板：
   - **新品上市**：用于推广新产品
   - **限时优惠**：用于限时促销活动
   - **满减活动**：用于满减优惠
   - **注册有礼**：用于新用户注册奖励
   - **转发有奖**：用于转发分享奖励
   - **招募代理**：用于招募代理商
   - **招工用工**：用于招聘员工
   - **招商加盟**：用于招商加盟
   - **节日促销**：用于节日营销
   - **会员活动**：用于会员专属活动

#### 2.2 编辑内容
1. **标题编辑**
   - 在标题输入框中输入营销标题
   - 注意字数限制（通常20-30字）
   - 系统会实时显示剩余字数

2. **详情内容**
   - 在详情编辑框中输入营销详情
   - 支持多行文本输入
   - 注意字数限制（通常200-500字）
   - 系统会实时显示字数统计

3. **上传图片**
   - 点击上传图片按钮
   - 选择要上传的图片文件
   - 支持多张图片上传
   - 图片会自动调整尺寸以适应界面
   - 可以预览已上传的图片

4. **上传视频**（可选）
   - 点击上传视频按钮
   - 选择要上传的视频文件
   - 支持常见视频格式（MP4、AVI等）

5. **预览内容**
   - 点击预览按钮查看效果
   - 可以随时返回编辑

#### 2.3 AI优化
1. 点击"AI优化"按钮
2. 系统自动对内容进行优化：
   - **文案优化**：提升文案吸引力和专业性
   - **排版优化**：优化内容排版结构
   - **表情符号优化**：添加合适的表情符号
   - **关键词优化**：优化关键词布局

3. 查看优化结果：
   - 左侧显示原文内容
   - 右侧显示优化后内容
   - 上下对比排列

4. 编辑优化内容：
   - 可以直接编辑优化后的内容
   - 支持修改标题和详情
   - 可以重新上传图片

5. 确认优化：
   - 满意后点击"下一步"
   - 进入渠道发布页面

### 3. 多渠道发布

#### 3.1 选择发布渠道
1. 进入渠道发布页面
2. 查看可用的发布渠道：
   - **小红书**：适合图文并茂的内容
   - **微信视频号**：适合短视频内容
   - **抖音**：适合短视频和直播
   - **微信朋友圈**：适合图文和短视频

3. 点击渠道图标查看该渠道的专属内容
4. 系统会自动为每个渠道生成适配的文案
5. 可以手动编辑各渠道的内容

#### 3.2 编辑渠道内容
1. 点击某个渠道图标
2. 在营销内容框中显示该渠道的内容
3. 可以编辑标题和详情
4. 注意各渠道的字数限制：
   - 小红书：标题20字，详情1000字
   - 微信视频号：标题30字，详情500字
   - 抖音：标题30字，详情200字
   - 微信朋友圈：标题20字，详情500字

5. 点击其他渠道图标切换内容
6. 各渠道内容独立编辑

#### 3.3 一键发布
1. 勾选要发布的渠道（可以多选）
2. 点击"一键发布"按钮
3. 系统检查必填项：
   - 标题不能为空
   - 详情内容不能为空
   - 至少选择一个发布渠道
   - 图片不能为空

4. 发布进度显示：
   - 显示正在发布的渠道
   - 显示发布进度百分比
   - 显示发布状态

5. 发布结果：
   - 成功：显示"发布成功"提示
   - 失败：显示失败原因

6. 发布后操作：
   - 可以查看发布历史
   - 可以创建新的营销内容
   - 可以查看数据统计

### 4. 店铺管理

#### 4.1 修改店铺信息
1. 点击店铺信息管理
2. 可以修改以下信息：
   - 店铺名称
   - 行业分类
   - 联系电话
   - 详细地址
   - 营业时间
   - 所在社区
   - 店铺logo
   - 门头照片

3. 修改后点击保存

#### 4.2 查看数据统计
1. 点击数据统计
2. 查看各项数据：
   - 发布次数：累计发布营销内容的次数
   - 浏览量：营销内容的总浏览次数
   - 收藏量：用户收藏营销内容的次数
   - 订单量：通过营销内容产生的订单数

3. 选择时间范围：
   - 今天
   - 本周
   - 本月
   - 自定义时间

4. 查看数据图表：
   - 折线图显示趋势
   - 柱状图显示对比

### 5. 发布管理

#### 5.1 查看发布历史
1. 点击发布历史
2. 查看所有已发布的营销内容
3. 可以按时间排序
4. 可以按渠道筛选

#### 5.2 草稿管理
1. 点击草稿箱
2. 查看未完成的营销内容
3. 可以继续编辑
4. 可以删除草稿

## 常见问题

### Q1：忘记密码怎么办？
A：目前系统支持手机号和微信登录，无需密码。如果手机号无法登录，请联系客服。

### Q2：如何修改店铺logo？
A：进入店铺信息管理页面，点击logo上传按钮，选择新的logo图片，点击保存即可。

### Q3：AI优化可以修改吗？
A：可以。AI优化后的内容可以手动编辑，编辑后可以再次进行AI优化。

### Q4：可以发布到哪些平台？
A：目前支持小红书、微信视频号、抖音、微信朋友圈四个平台。

### Q5：发布失败怎么办？
A：请检查：
1. 网络连接是否正常
2. 内容是否符合平台规范
3. 图片和视频格式是否正确
如果问题仍然存在，请联系客服。

### Q6：如何查看营销效果？
A：进入数据统计页面，可以查看发布次数、浏览量、收藏量、订单量等数据。

### Q7：居民端需要注册吗？
A：目前居民端无需注册即可浏览营销信息，但收藏和下单功能需要登录。

### Q8：如何联系客服？
A：可以通过以下方式联系客服：
- 电话：400-XXX-XXXX
- 微信：客服微信号
- 邮箱：service@linmeng.com

## 使用技巧

### 商家端技巧

#### 1. 内容创作技巧
- **标题吸引人**：使用数字、疑问句、感叹句等吸引注意
- **内容有价值**：提供优惠信息、实用技巧、产品介绍等
- **图片质量高**：使用高清图片，展示产品细节
- **视频生动**：短视频展示产品使用场景

#### 2. AI优化技巧
- 先写好基础内容，再进行AI优化
- 优化后可以手动调整，保持品牌调性
- 不同渠道可以有不同的优化策略
- 定期优化历史内容，保持新鲜感

#### 3. 多渠道发布技巧
- 根据内容特点选择合适的渠道
- 不同渠道使用不同的文案风格
- 合理安排发布时间，避开高峰期
- 定期分析各渠道效果，优化发布策略

#### 4. 数据分析技巧
- 定期查看数据统计，了解营销效果
- 分析高浏览量的内容特点
- 关注收藏和转化数据
- 根据数据调整营销策略

### 居民端技巧

#### 1. 发现优惠技巧
- 定期浏览营销信息，发现新优惠
- 关注喜欢的商家，第一时间获取优惠
- 使用收藏功能，方便后续查看
- 分享优惠给朋友，获得更多福利

#### 2. 下单技巧
- 仔细阅读营销内容，了解优惠详情
- 确认商家营业时间，避免白跑一趟
- 下单前联系商家确认库存

## 注意事项

### 商家端注意事项

1. **内容规范**
   - 发布真实有效的营销信息
   - 不发布虚假宣传内容
   - 遵守各平台的内容规范
   - 不发布违法违规内容

2. **图片要求**
   - 使用原创图片或授权图片
   - 图片清晰度要高
   - 不使用低俗、暴力图片
   - 图片大小建议不超过5MB

3. **发布频率**
   - 合理控制发布频率
   - 避免频繁发布相似内容
   - 保持内容的新鲜感和多样性

4. **数据安全**
   - 保护用户隐私信息
   - 不泄露客户联系方式
   - 定期备份重要数据

### 居民端注意事项

1. **信息核实**
   - 仔细阅读营销内容
   - 核实优惠活动的真实性
   - 有疑问及时联系商家

2. **个人信息保护**
   - 谨慎提供个人信息
   - 不轻易透露银行卡信息
   - 保护好账号密码

3. **交易安全**
   - 选择正规商家
   - 优先使用平台支付
   - 保留交易凭证
