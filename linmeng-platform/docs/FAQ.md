## 一、注册登录问题

### Q1: 如何注册成为社区用户？
**A:** 
1. 访问统一入口页面
2. 点击"社区端"
3. 在登录页点击"立即注册"
4. 填写区域、负责人姓名、手机号、社区名称等信息
5. 获取验证码并提交
6. 等待管理员审核通过

### Q2: 如何注册成为商家用户？
**A:**
1. 访问统一入口页面
2. 点击"商家端"
3. 在登录页点击"立即注册"
4. 填写联系人姓名、手机号、商家名称、经营类型、地址等信息
5. 获取验证码并提交
6. 等待管理员审核通过

### Q3: 如何注册成为招商大使？
**A:**
1. 访问招商大使端：http://localhost:5173/
2. 点击"注册"
3. 填写姓名、手机号
4. 获取验证码
5. 设置密码
6. 提交注册即可使用

### Q4: 验证码收不到怎么办？
**A:**
- 确认手机号输入正确
- 检查手机是否设置了短信拦截
- 验证码会自动填入输入框，请查看页面弹窗提示
- 如仍无法收到，请联系客服

### Q5: 登录时提示"账号待审核"怎么办？
**A:**
社区用户和商家用户注册后需要管理员审核，请耐心等待。审核通过后会收到系统通知。

### Q6: 忘记密码怎么办？
**A:**
目前系统采用验证码登录方式，无需密码。如需密码登录，请联系管理员。

## 二、活动管理问题

### Q7: 如何发布活动？
**A:**
1. 登录社区端
2. 点击底部导航"发布活动"
3. 填写活动标题、类型、时间、地点、人数、描述、所需资源等信息
4. 点击"提交"

### Q8: 活动发布后多久能通过审核？
**A:**
一般情况下，活动会在1-2个工作日内审核完成。审核结果会通过消息中心通知。

### Q9: 如何修改已发布的活动？
**A:**
1. 进入"我的活动"列表
2. 点击需要修改的活动
3. 点击"编辑"按钮
4. 修改后保存提交

### Q10: 活动完成后需要做什么？
**A:**
活动结束后，请及时在系统中确认完成：
1. 进入活动详情页
2. 点击"完成活动"
3. 填写活动总结（可选）
4. 确认提交

## 三、赞助管理问题

### Q11: 如何发布赞助信息？
**A:**
1. 登录商家端
2. 点击底部导航"发布赞助"
3. 填写赞助标题、类型、资源类型、价值、描述、有效期等信息
4. 点击"提交"

### Q12: 赞助发布后多久能通过审核？
**A:**
一般情况下，赞助信息会在1-2个工作日内审核完成。

### Q13: 如何下架赞助？
**A:**
1. 进入"我的赞助"列表
2. 点击需要下架的赞助
3. 点击"下架"按钮

### Q14: 赞助被拒绝怎么办？
**A:**
查看拒绝原因，修改后重新提交审核。常见拒绝原因：
- 信息不完整
- 内容不符合规范
- 联系方式无效

## 四、合作对接问题

### Q15: 如何申请赞助？
**A:**
1. 登录社区端
2. 进入"商家赞助"列表
3. 找到合适的赞助
4. 点击查看详情
5. 点击"申请合作"
6. 填写合作意向

### Q16: 如何查看合作申请状态？
**A:**
进入"合作"页面，可以查看所有合作记录及其状态：
- 待确认：等待对方确认
- 已确认：合作已达成
- 已完成：活动已结束
- 已拒绝：合作被拒绝

### Q17: 如何取消合作申请？
**A:**
在合作详情页，如果状态为"待确认"，可以点击"取消申请"。

### Q18: 合作完成后如何评价？
**A:**
1. 进入合作详情页
2. 点击"评价商家"
3. 选择评分（1-5星）
4. 填写评价内容
5. 提交评价

## 五、招商大使问题

### Q19: 招商大使的提成比例是多少？
**A:**
默认提成比例为10%，具体以平台规则为准。

### Q20: 提成什么时候结算？
**A:**
提成在商家首次付费成功后自动生成，结算周期以平台规则为准。

### Q21: 如何获取推广链接？
**A:**
1. 登录招商大使端
2. 进入"个人中心"
3. 查看专属渠道码和推广链接

### Q22: 推广链接可以分享给谁？
**A:**
推广链接可以分享给任何有意向入驻平台的商家。商家通过您的链接注册后，您将获得相应的提成。

## 六、消息通知问题

### Q23: 如何查看消息？
**A:**
点击首页右上角的消息图标（铃铛），进入消息中心查看所有消息。

### Q24: 消息类型有哪些？
**A:**
- **对接消息**：合作申请、合作确认等
- **系统通知**：审核结果、活动提醒等
- **留言消息**：用户留言咨询

### Q25: 如何回复留言？
**A:**
1. 进入消息中心
2. 点击留言消息
3. 在详情页填写回复内容
4. 点击"发送"

### Q26: 如何标记消息已读？
**A:**
- 点击消息后自动标记已读
- 或点击"全部已读"按钮

## 八、系统使用问题

### Q31: 页面打不开怎么办？
**A:**
- 检查网络连接
- 清除浏览器缓存
- 尝试更换浏览器
- 联系技术支持

### Q32: 数据提交失败怎么办？
**A:**
- 检查必填项是否完整
- 检查网络连接
- 刷新页面重试
- 联系技术支持

### Q33: 图片上传失败怎么办？
**A:**
- 检查图片格式（支持jpg、png）
- 检查图片大小（不超过5MB）
- 检查网络连接
- 尝试压缩图片后上传

### Q34: 智能匹配不准确怎么办？
**A:**
系统会根据活动需求和赞助信息进行智能匹配，如匹配结果不满意，可以：
- 完善活动/赞助信息
- 手动筛选查找
- 联系客服协助
