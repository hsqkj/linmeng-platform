## 三、组件库建设

### 1. 匹配度展示组件
**文件位置**：
- `apps/community-h5/src/components/MatchScore.vue`
- `apps/merchant-h5/src/components/MatchScore.vue`

**功能特点**：
- 使用5颗心形图标展示匹配度
- 支持动画效果（心跳动画）
- 自动根据分数计算心形数量
- 响应式设计

**使用示例**：
```vue
<MatchScore :score="85" />
```

### 2. 空状态组件
**文件位置**：
- `apps/community-h5/src/components/EmptyState.vue`
- `apps/merchant-h5/src/components/EmptyState.vue`

**功能特点**：
- 可自定义图标、文本、描述
- 支持操作按钮
- 淡入动画效果
- 响应式设计

**使用示例**：
```vue
<EmptyState
  icon="todo-list-o"
  text="暂无数据"
  description="点击下方按钮添加"
  buttonText="立即添加"
  @action="handleAdd"
/>
```

### 3. 骨架屏组件
**文件位置**：
- `apps/community-h5/src/components/Skeleton.vue`
- `apps/merchant-h5/src/components/Skeleton.vue`

**功能特点**：
- 支持多种类型：default、card、list、stat
- 可自定义行数
- 淡入动画效果
- 提升用户体验

**使用示例**：
```vue
<Skeleton type="card" />
<Skeleton type="list" :rows="5" />
<Skeleton type="stat" />
```

## 七、技术实现细节

### 1. CSS变量使用
```css
/* 推荐 */
background: var(--primary-color);
padding: var(--spacing-md);
border-radius: var(--border-radius-lg);

/* 不推荐 */
background: #4facfe;
padding: 16px;
border-radius: 12px;
```

### 2. 组件引入
```vue
<script setup>
import MatchScore from '@/components/MatchScore.vue'
import EmptyState from '@/components/EmptyState.vue'
import Skeleton from '@/components/Skeleton.vue'
</script>
```

### 3. 动画使用
```css
/* 淡入动画 */
.fade-in {
  animation: fadeIn 0.3s ease;
}

/* 心跳动画 */
@keyframes heartBeat {
  0% { transform: scale(1); }
  50% { transform: scale(1.3); }
  100% { transform: scale(1); }
}
```
