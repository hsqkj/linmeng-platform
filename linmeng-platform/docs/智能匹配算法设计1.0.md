# 社区-商家智能匹配算法设计文档

## 一、算法设计目标

### 1.1 核心目标
- **精准匹配**：提高匹配成功率，降低无效推荐
- **高效推荐**：快速筛选出最匹配的商家
- **双向满意**：兼顾社区需求和商家期望
- **持续优化**：基于历史数据不断优化算法

### 1.2 匹配场景
1. **社区发布需求** → 推荐匹配商家
2. **商家发布赞助** → 推荐匹配需求
3. **商家浏览需求** → 按匹配度排序展示

## 二、匹配维度设计

### 2.1 核心匹配维度（权重70%）

#### 2.1.1 赞助类型匹配（权重25%）
**匹配逻辑**：
```
商家可提供赞助类型 ∩ 需求赞助类型
```

**类型枚举**：
- 现金赞助
- 物资赞助
- 人力支持
- 场地提供
- 媒体宣传
- 技术支持
- 专业服务

**匹配规则**：
- 完全匹配：商家可提供类型包含需求所有类型 → 得分100%
- 部分匹配：商家可提供类型包含需求部分类型 → 得分 = 匹配类型数 / 需求类型数 × 100%
- 无匹配：无交集 → 得分0%

**示例**：
```
需求：现金,物资,人力
商家：现金,物资,场地

匹配类型：现金,物资（2个）
得分：(2/3) × 100% = 66.7%
```

#### 2.1.2 目标人群匹配（权重25%）
**匹配逻辑**：
```
商家目标人群 ∩ 需求面向人群
```

**人群标签**：
- 亲子/少儿
- 中老年
- 青年白领
- 宝妈
- 全社区居民
- 老年人
- 青少年

**匹配规则**：
- 完全匹配：商家目标人群包含需求所有人群 → 得分100%
- 部分匹配：商家目标人群包含需求部分人群 → 得分 = 匹配人群数 / 需求人群数 × 100%
- 包含匹配：商家目标人群为"全社区居民" → 得分80%
- 无匹配：无交集 → 得分0%

#### 2.1.3 行业领域匹配（权重20%）
**匹配逻辑**：
```
商家所属行业 ∩ 需求相关行业
```

**行业分类**：
- 餐饮美食
- 生活服务
- 教育培训
- 医疗健康
- 文化娱乐
- 零售购物
- 金融服务
- 房产家居
- 运动健身
- 美容美发

**匹配规则**：
- 直接匹配：商家行业与需求行业一致 → 得分100%
- 相关匹配：商家行业与需求行业相关 → 得分60%
- 无匹配：不相关 → 得分0%

### 2.2 辅助匹配维度（权重20%）

#### 2.2.1 地理位置匹配（权重10%）
**匹配逻辑**：
```
计算商家与社区的距离
```

**距离计算**：
使用Haversine公式计算经纬度距离

**匹配规则**：
- 距离 ≤ 1km → 得分100%
- 1km < 距离 ≤ 3km → 得分80%
- 3km < 距离 ≤ 5km → 得分60%
- 5km < 距离 ≤ 10km → 得分40%
- 距离 > 10km → 得分20%

#### 2.2.2 社区特征匹配（权重10%）
**匹配逻辑**：
```
社区人口结构与商家目标人群匹配度
```

**匹配规则**：
- 老年人占比高 + 商家目标人群包含"中老年" → 加分
- 青少年家庭占比高 + 商家目标人群包含"亲子/少儿" → 加分
- 社区规模大 + 商家服务能力强 → 加分

### 2.3 质量加分维度（权重10%）

#### 2.3.1 商家信誉评分（权重5%）
**评分因素**：
- 星级评分（5分制）
- 历史合作成功率
- 用户评价数量
- 投诉率

**计算公式**：
```
信誉得分 = 星级评分 × 20% + 合作成功率 × 50% + 评价权重 × 30%
```

#### 2.3.2 会员等级加分（权重3%）
**加分规则**：
- 钻石会员 → +3分
- 金卡会员 → +2分
- 银卡会员 → +1分
- 普通会员 → +0分

#### 2.3.3 标签匹配加分（权重2%）
**匹配逻辑**：
```
商家标签 ∩ 需求标签
```

**计算规则**：
- 每匹配一个标签 → +0.5分
- 最高不超过2分

## 三、综合匹配算法

### 3.1 总分计算公式

```
总分 = 核心维度得分 × 70% + 辅助维度得分 × 20% + 质量加分 × 10%
```

**详细公式**：
```
总分 = 
  (赞助类型匹配 × 25% + 目标人群匹配 × 25% + 行业匹配 × 20%) × 70% +
  (地理位置匹配 × 10% + 社区特征匹配 × 10%) × 20% +
  (信誉评分 × 5% + 会员等级 × 3% + 标签匹配 × 2%) × 10%
```

### 3.2 匹配等级划分

| 匹配度 | 等级 | 推荐策略 |
|--------|------|----------|
| 90-100% | ⭐⭐⭐⭐⭐ 极高匹配 | 强烈推荐，优先推送 |
| 80-89% | ⭐⭐⭐⭐ 高度匹配 | 重点推荐，置顶展示 |
| 70-79% | ⭐⭐⭐ 中度匹配 | 正常推荐，排序靠前 |
| 60-69% | ⭐⭐ 基本匹配 | 一般推荐，正常展示 |
| 50-59% | ⭐ 低度匹配 | 仅在搜索结果中展示 |
| <50% | 不匹配 | 不推荐 |

### 3.3 匹配结果展示

**展示形式**：
- 使用5颗红心表示匹配度
- 每颗红心代表20%的匹配度
- 红心旁边显示"匹配度：⭐⭐⭐⭐⭐"

**计算规则**：
```
红心数量 = Math.ceil(匹配度 / 20)
```

**示例**：
- 匹配度95% → 5颗红心 ⭐⭐⭐⭐⭐
- 匹配度82% → 5颗红心 ⭐⭐⭐⭐⭐
- 匹配度75% → 4颗红心 ⭐⭐⭐⭐
- 匹配度65% → 4颗红心 ⭐⭐⭐⭐
- 匹配度55% → 3颗红心 ⭐⭐⭐

## 四、算法实现

### 4.1 核心算法代码

```javascript
class MatchingAlgorithm {
  /**
   * 计算商家与需求的匹配度
   * @param {Object} merchant - 商家信息
   * @param {Object} activity - 需求信息
   * @param {Object} community - 社区信息（可选）
   * @returns {Object} 匹配结果 { score, details }
   */
  static calculateMatchScore(merchant, activity, community = null) {
    const details = {}
    
    // 1. 赞助类型匹配（25%）
    details.sponsorTypeScore = this.matchSponsorTypes(
      merchant.sponsor_types,
      activity.sponsor_types
    )
    
    // 2. 目标人群匹配（25%）
    details.targetCrowdScore = this.matchTargetCrowd(
      merchant.target_crowd,
      activity.target_crowd
    )
    
    // 3. 行业领域匹配（20%）
    details.industryScore = this.matchIndustry(
      merchant.industry,
      activity.sponsor_types,
      activity.demand_type
    )
    
    // 4. 地理位置匹配（10%）
    details.locationScore = 0
    if (community && merchant.latitude && merchant.longitude && 
        community.latitude && community.longitude) {
      details.locationScore = this.matchLocation(
        merchant.latitude,
        merchant.longitude,
        community.latitude,
        community.longitude
      )
    }
    
    // 5. 社区特征匹配（10%）
    details.communityScore = 0
    if (community) {
      details.communityScore = this.matchCommunity(
        community,
        merchant.target_crowd
      )
    }
    
    // 6. 商家信誉评分（5%）
    details.reputationScore = this.calculateReputation(merchant)
    
    // 7. 会员等级加分（3%）
    details.memberScore = this.getMemberScore(merchant.member_level)
    
    // 8. 标签匹配加分（2%）
    details.tagScore = this.matchTags(
      merchant.tags,
      merchant.custom_tags,
      activity.tags,
      activity.custom_tags
    )
    
    // 计算总分
    const coreScore = (
      details.sponsorTypeScore * 25 +
      details.targetCrowdScore * 25 +
      details.industryScore * 20
    ) / 70
    
    const assistScore = (
      details.locationScore * 10 +
      details.communityScore * 10
    ) / 20
    
    const qualityScore = (
      details.reputationScore * 5 +
      details.memberScore * 3 +
      details.tagScore * 2
    ) / 10
    
    const totalScore = Math.round(
      coreScore * 70 + 
      assistScore * 20 + 
      qualityScore * 10
    )
    
    return {
      score: totalScore,
      hearts: Math.ceil(totalScore / 20),
      details
    }
  }
  
  /**
   * 赞助类型匹配
   */
  static matchSponsorTypes(merchantTypes, activityTypes) {
    if (!merchantTypes || !activityTypes) return 0
    
    const merchant = merchantTypes.split(',').map(t => t.trim())
    const activity = activityTypes.split(',').map(t => t.trim())
    
    const matched = merchant.filter(t => activity.includes(t))
    
    return activity.length > 0 ? (matched.length / activity.length) * 100 : 0
  }
  
  /**
   * 目标人群匹配
   */
  static matchTargetCrowd(merchantCrowd, activityCrowd) {
    if (!merchantCrowd || !activityCrowd) return 0
    
    const merchant = merchantCrowd.split(',').map(t => t.trim())
    const activity = activityCrowd.split(',').map(t => t.trim())
    
    // 如果商家目标人群是"全社区居民"，给予80%得分
    if (merchant.includes('全社区居民')) {
      return 80
    }
    
    const matched = merchant.filter(t => activity.includes(t))
    
    return activity.length > 0 ? (matched.length / activity.length) * 100 : 0
  }
  
  /**
   * 行业领域匹配
   */
  static matchIndustry(merchantIndustry, activityTypes, demandType) {
    if (!merchantIndustry) return 0
    
    const industry = merchantIndustry.split(',').map(t => t.trim())
    const types = activityTypes ? activityTypes.split(',').map(t => t.trim()) : []
    
    // 行业关联映射
    const industryMapping = {
      '餐饮美食': ['物资', '场地'],
      '教育培训': ['人力', '专业服务', '场地'],
      '医疗健康': ['人力', '专业服务', '物资'],
      '文化娱乐': ['场地', '物资', '人力'],
      '运动健身': ['场地', '物资', '人力'],
      '生活服务': ['人力', '物资', '专业服务'],
      '零售购物': ['物资', '现金'],
      '美容美发': ['人力', '物资', '专业服务']
    }
    
    let maxScore = 0
    
    for (const ind of industry) {
      // 直接匹配
      if (types.includes(ind)) {
        maxScore = Math.max(maxScore, 100)
        continue
      }
      
      // 相关匹配
      const relatedTypes = industryMapping[ind] || []
      const matchedTypes = relatedTypes.filter(t => types.includes(t))
      if (matchedTypes.length > 0) {
        maxScore = Math.max(maxScore, 60)
      }
    }
    
    return maxScore
  }
  
  /**
   * 地理位置匹配
   */
  static matchLocation(lat1, lon1, lat2, lon2) {
    const distance = this.calculateDistance(lat1, lon1, lat2, lon2)
    
    if (distance <= 1) return 100
    if (distance <= 3) return 80
    if (distance <= 5) return 60
    if (distance <= 10) return 40
    return 20
  }
  
  /**
   * 计算两点间距离（km）
   */
  static calculateDistance(lat1, lon1, lat2, lon2) {
    const R = 6371 // 地球半径（km）
    const dLat = this.toRad(lat2 - lat1)
    const dLon = this.toRad(lon2 - lon1)
    const a = 
      Math.sin(dLat/2) * Math.sin(dLat/2) +
      Math.cos(this.toRad(lat1)) * Math.cos(this.toRad(lat2)) *
      Math.sin(dLon/2) * Math.sin(dLon/2)
    const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a))
    return R * c
  }
  
  static toRad(deg) {
    return deg * (Math.PI / 180)
  }
  
  /**
   * 社区特征匹配
   */
  static matchCommunity(community, merchantCrowd) {
    if (!community || !merchantCrowd) return 0
    
    let score = 50 // 基础分
    
    const crowd = merchantCrowd.split(',').map(t => t.trim())
    
    // 老年人占比高
    if (community.elderly_ratio > 30 && crowd.includes('中老年')) {
      score += 20
    }
    
    // 青少年家庭占比高
    if (community.youth_family_ratio > 40 && crowd.includes('亲子/少儿')) {
      score += 20
    }
    
    // 社区规模大
    if (community.total_households > 2000) {
      score += 10
    }
    
    return Math.min(score, 100)
  }
  
  /**
   * 商家信誉评分
   */
  static calculateReputation(merchant) {
    let score = 50 // 基础分
    
    // 星级评分
    if (merchant.star_rating) {
      score += (merchant.star_rating / 5) * 30
    }
    
    // 会员等级加分（已在其他维度计算，这里不重复）
    
    return Math.min(score, 100)
  }
  
  /**
   * 会员等级加分
   */
  static getMemberScore(memberLevel) {
    const scores = {
      3: 100, // 钻石
      2: 80,  // 金卡
      1: 60,  // 银卡
      0: 40   // 普通
    }
    return scores[memberLevel] || 40
  }
  
  /**
   * 标签匹配
   */
  static matchTags(merchantTags, merchantCustomTags, activityTags, activityCustomTags) {
    let matchedCount = 0
    
    // 解析标签
    const mTags = this.parseTags(merchantTags, merchantCustomTags)
    const aTags = this.parseTags(activityTags, activityCustomTags)
    
    // 计算匹配数量
    for (const tag of mTags) {
      if (aTags.includes(tag)) {
        matchedCount++
      }
    }
    
    // 每匹配一个标签得10分，最高100分
    return Math.min(matchedCount * 10, 100)
  }
  
  /**
   * 解析标签
   */
  static parseTags(tags, customTags) {
    let result = []
    
    if (tags) {
      try {
        const tagIds = JSON.parse(tags)
        result = result.concat(tagIds.map(id => String(id)))
      } catch (e) {}
    }
    
    if (customTags) {
      try {
        const custom = JSON.parse(customTags)
        result = result.concat(custom.map(t => t.name || t))
      } catch (e) {}
    }
    
    return result
  }
  
  /**
   * 批量计算匹配度并排序
   */
  static async matchAndSort(merchants, activity, community = null) {
    const results = merchants.map(merchant => {
      const matchResult = this.calculateMatchScore(merchant, activity, community)
      return {
        merchant,
        ...matchResult
      }
    })
    
    // 按匹配度降序排序
    results.sort((a, b) => b.score - a.score)
    
    return results
  }
  
  /**
   * 筛选高匹配度商家
   */
  static filterHighMatch(results, threshold = 70) {
    return results.filter(r => r.score >= threshold)
  }
}

module.exports = MatchingAlgorithm
```

### 4.2 使用示例

```javascript
const MatchingAlgorithm = require('../utils/matchingAlgorithm')

// 社区发布需求时，推荐匹配商家
router.post('/create', authMiddleware, async (req, res) => {
  // ... 创建需求
  
  // 获取所有活跃商家
  const merchants = await MerchantUser.findAll({
    where: { status: 1 }
  })
  
  // 获取社区信息
  const community = await CommunityUser.findByPk(id)
  
  // 计算匹配度并排序
  const matchResults = MatchingAlgorithm.matchAndSort(
    merchants, 
    activity, 
    community
  )
  
  // 筛选高匹配度商家（≥70分）
  const highMatchMerchants = MatchingAlgorithm.filterHighMatch(matchResults, 70)
  
  // 推送消息给高匹配度商家
  if (highMatchMerchants.length > 0) {
    const messages = highMatchMerchants.map(({ merchant, score }) => ({
      merchant_id: merchant.id,
      title: '新需求推荐',
      content: `匹配度${score}%！${community.community_name}发布了新需求"${activity.title}"，与您的业务高度匹配，快去看看吧！`,
      type: 'demand_recommend',
      related_id: activity.id,
      is_read: 0
    }))
    
    await Message.bulkCreate(messages)
  }
  
  // 返回结果
  res.json({
    code: 200,
    message: '需求发布成功',
    data: {
      activity,
      matchCount: highMatchMerchants.length,
      topMatches: matchResults.slice(0, 5)
    }
  })
})
```
