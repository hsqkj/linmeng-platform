## 一、产品简介

### 1.1 产品概述
邻盟是一款连接社区与商家的智能资源匹配平台。社区可以通过平台发布活动需求，获取商家的赞助支持；商家可以通过平台精准触达社区居民，提升品牌形象。

### 1.2 核心价值
- **社区端**：免费获取商家赞助、物资、人力、场地、技术、专家、媒体宣传等优质爱心资源
- **商家端**：精准触达社区居民，提升品牌信用，高效获客转化
- **招商大使**：推广商家入驻，获取提成收益

### 1.3 适用人群
| 角色 | 适用对象 |
|-----|---------|
| 社区用户 | 社区居委会、社区工作者、社区志愿者 |
| 商家用户 | 周边商户、企业、公益组织 |
| 招商大使 | 推广人员、社区合伙人 |

## 二、快速开始

### 2.1 访问入口
**统一入口地址**：http://localhost:5170/

### 2.2 选择身份
打开统一入口页面后，根据您的身份选择：
- **社区端**：社区用户点击进入
- **商家端**：商家用户点击进入

### 2.3 注册流程

#### 社区用户注册
1. 点击"立即注册"
2. 选择所在区域（区/街道/社区）
3. 填写负责人姓名
4. 填写手机号并获取验证码
5. 填写社区名称
6. 提交注册，等待审核

#### 商家用户注册
1. 点击"立即注册"
2. 填写联系人姓名
3. 填写手机号并获取验证码
4. 填写商家名称
5. 选择经营类型
6. 填写商家地址
7. 提交注册，等待审核

#### 招商大使注册
1. 访问招商大使端：http://localhost:5173/
2. 点击"注册"
3. 填写姓名
4. 填写手机号并获取验证码
5. 设置密码
6. 提交注册

## 三、社区端使用指南

### 3.1 登录
1. 访问统一入口，选择"社区端"
2. 输入手机号
3. 点击"发送验证码"
4. 输入验证码（自动填入）
5. 点击"登录"

### 3.2 发布活动

#### 3.2.1 进入发布页面
登录后，点击底部导航"发布活动"或首页"发布活动"按钮

#### 3.2.2 填写活动信息
| 字段 | 说明 | 是否必填 |
|-----|------|---------|
| 活动标题 | 活动名称 | 是 |
| 活动类型 | 如节日活动、公益活动等 | 是 |
| 活动时间 | 开始和结束时间 | 是 |
| 活动地点 | 活动举办地点 | 是 |
| 预计人数 | 参与人数预估 | 是 |
| 活动描述 | 详细描述活动内容 | 是 |
| 所需资源 | 需要的赞助资源 | 是 |
| 活动图片 | 上传活动相关图片 | 否 |

#### 3.2.3 提交审核
填写完成后点击"提交"，活动将进入审核状态

### 3.3 浏览赞助

#### 3.3.1 查看赞助列表
点击底部导航"赞助"或首页"商家赞助"入口

#### 3.3.2 筛选赞助
- 按类型筛选：物资、资金、服务、场地等
- 按区域筛选：附近商家优先

#### 3.3.3 查看赞助详情
点击赞助卡片，查看详细信息：
- 赞助内容
- 资源价值
- 商家信息
- 适用条件

#### 3.3.4 申请赞助
在赞助详情页点击"申请合作"，填写合作意向

### 3.4 管理合作

#### 3.4.1 查看合作列表
点击底部导航"合作"，查看所有合作记录

#### 3.4.2 处理合作申请
- **接受合作**：确认接受商家的合作申请
- **拒绝合作**：填写拒绝原因后拒绝

#### 3.4.3 完成合作
活动结束后，在合作详情页点击"完成合作"

### 3.5 评价商家

#### 3.5.1 进入评价页面
合作完成后，在合作详情页点击"评价商家"

#### 3.5.2 填写评价
- 评分：1-5星
- 评价内容：描述合作体验
- 提交评价

### 3.6 消息中心

#### 3.6.1 查看消息
点击首页右上角消息图标，进入消息中心

#### 3.6.2 消息类型
| 类型 | 说明 |
|-----|------|
| 对接消息 | 商家合作申请通知 |
| 系统通知 | 审核结果、活动提醒等 |
| 留言消息 | 商家留言回复 |

#### 3.6.3 处理消息
- 点击消息查看详情
- 点击"标记已读"

### 3.7 个人中心

#### 3.7.1 查看个人信息
点击底部导航"我的"，查看个人资料

#### 3.7.2 编辑资料
点击"编辑资料"，修改个人信息

#### 3.7.3 查看我的活动
点击"我的活动"，查看发布的所有活动

## 五、招商大使使用指南

### 5.1 注册登录

#### 5.1.1 注册
1. 访问招商大使端：http://localhost:5173/
2. 点击"注册"
3. 填写姓名、手机号
4. 获取验证码
5. 设置密码
6. 提交注册

#### 5.1.2 登录
1. 输入手机号和密码
2. 点击"登录"

### 5.2 推广商家

#### 5.2.1 获取推广链接
登录后，在"个人中心"查看专属推广链接

#### 5.2.2 推广方式
- 分享链接给商家
- 生成二维码让商家扫码注册

#### 5.2.3 推广链接格式
```
http://localhost:5170/?channel_code=您的渠道码
```

### 5.3 查看推广商家

#### 5.3.1 商家列表
在"商家列表"查看通过您推广注册的商家

#### 5.3.2 商家状态
- 待审核：商家注册待审核
- 已通过：商家已入驻成功
- 已禁用：商家账号被禁用

### 5.4 提成管理

#### 5.4.1 查看提成
在"我的提成"查看所有提成记录

#### 5.4.2 提成明细
| 字段 | 说明 |
|-----|------|
| 商家名称 | 产生提成的商家 |
| 订单金额 | 商家付费金额 |
| 提成比例 | 提成百分比 |
| 提成金额 | 实际提成金额 |
| 状态 | 待结算/已结算 |

#### 5.4.3 提成规则
- 商家首次付费后自动计算提成
- 提成比例默认为10%
- 提成金额在商家付费成功后生成
