## 四、使用说明

### 1. 社区端留言流程
1. 社区用户登录后查看商家赞助列表
2. 点击赞助项查看详情
3. 在详情页底部留言
4. 商家收到留言提醒
5. 商家回复留言
6. 社区用户查看回复

### 2. 业务员推广流程
1. 业务员注册账号
2. 登录后获取专属渠道码和链接
3. 分享渠道码/链接给商家
4. 商家通过渠道码注册
5. 商家缴费后自动计算提成
6. 业务员查看提成统计

### 3. 商家注册流程（带渠道码）
1. 业务员分享渠道链接
2. 商家点击链接打开注册页（自动填充渠道码）
3. 商家填写注册信息
4. 注册成功后自动绑定业务员
