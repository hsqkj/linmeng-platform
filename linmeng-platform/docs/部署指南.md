## 一、部署前准备

### 1.1 服务器要求
| 配置项 | 最低要求 | 推荐配置 |
|-------|---------|---------|
| CPU | 2核 | 4核 |
| 内存 | 4GB | 8GB |
| 硬盘 | 50GB | 100GB SSD |
| 带宽 | 3Mbps | 10Mbps |
| 操作系统 | CentOS 7+ / Ubuntu 18+ | Ubuntu 20.04 LTS |

### 1.2 所需软件
- Node.js 18+
- MySQL 8.0+
- Nginx
- PM2 (进程管理)
- Git

### 1.3 域名准备
- 需要一个已备案的域名
- 建议配置子域名：
  - `api.yourdomain.com` - 后端API
  - `www.yourdomain.com` - 统一入口
  - `admin.yourdomain.com` - 管理后台

## 二、服务器环境配置

### 2.1 安装Node.js
```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# CentOS/RHEL
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo yum install -y nodejs

# 验证安装
node -v
npm -v
```

### 2.2 安装MySQL
```bash
# Ubuntu
sudo apt update
sudo apt install mysql-server
sudo mysql_secure_installation

# CentOS
sudo yum install mysql-server
sudo systemctl start mysqld
sudo mysql_secure_installation

# 创建数据库
mysql -u root -p
CREATE DATABASE linmeng CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'linmeng'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON linmeng.* TO 'linmeng'@'localhost';
FLUSH PRIVILEGES;
```

### 2.3 安装Nginx
```bash
# Ubuntu
sudo apt install nginx

# CentOS
sudo yum install nginx

# 启动Nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

### 2.4 安装PM2
```bash
sudo npm install -g pm2
```

## 三、代码部署

### 3.1 克隆代码
```bash
# 创建项目目录
sudo mkdir -p /var/www/linmeng
sudo chown $USER:$USER /var/www/linmeng

# 克隆代码
cd /var/www
git clone your_repository_url linmeng
cd linmeng/linmeng-platform
```

### 3.2 安装依赖
```bash
# 安装根目录依赖
npm install

# 安装各端依赖
cd apps/portal && npm install && cd ../..
cd apps/community-h5 && npm install && cd ../..
cd apps/merchant-h5 && npm install && cd ../..
cd apps/salesman-h5 && npm install && cd ../..
cd apps/admin-web && npm install && cd ../..
cd server && npm install && cd ..
```

### 3.3 配置环境变量
```bash
# 创建服务端环境配置
cd server
cp .env.example .env.production
```

编辑 `.env.production`：
```env
NODE_ENV=production
PORT=3001
DB_HOST=localhost
DB_PORT=3306
DB_NAME=linmeng
DB_USER=linmeng
DB_PASSWORD=your_password
JWT_SECRET=your_jwt_secret_key_here
JWT_EXPIRES_IN=7d
```

### 3.4 数据库迁移
```bash
cd server
npx sequelize-cli db:migrate --env production
```

## 五、Nginx配置

### 5.1 创建Nginx配置
```bash
sudo nano /etc/nginx/sites-available/linmeng
```

配置内容：
```nginx
# 后端API
server {
    listen 80;
    server_name api.yourdomain.com;

    location / {
        proxy_pass http://127.0.0.1:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}

# 统一入口
server {
    listen 80;
    server_name yourdomain.com www.yourdomain.com;

    root /var/www/linmeng/linmeng-platform/apps/portal/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}

# 社区端
server {
    listen 80;
    server_name community.yourdomain.com;

    root /var/www/linmeng/linmeng-platform/apps/community-h5/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}

# 商家端
server {
    listen 80;
    server_name merchant.yourdomain.com;

    root /var/www/linmeng/linmeng-platform/apps/merchant-h5/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}

# 招商大使端
server {
    listen 80;
    server_name salesman.yourdomain.com;

    root /var/www/linmeng/linmeng-platform/apps/salesman-h5/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}

# 管理后台
server {
    listen 80;
    server_name admin.yourdomain.com;

    root /var/www/linmeng/linmeng-platform/apps/admin-web/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

### 5.2 启用配置
```bash
# 创建软链接
sudo ln -s /etc/nginx/sites-available/linmeng /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重载Nginx
sudo systemctl reload nginx
```

## 六、SSL证书配置 (HTTPS)

### 6.1 安装Certbot
```bash
# Ubuntu
sudo apt install certbot python3-certbot-nginx

# CentOS
sudo yum install certbot python3-certbot-nginx
```

### 6.2 申请证书
```bash
# 为所有域名申请证书
sudo certbot --nginx -d yourdomain.com -d www.yourdomain.com -d api.yourdomain.com -d community.yourdomain.com -d merchant.yourdomain.com -d salesman.yourdomain.com -d admin.yourdomain.com
```

### 6.3 自动续期
```bash
# 测试续期
sudo certbot renew --dry-run

# Certbot会自动添加定时任务
```

## 七、启动服务

### 7.1 使用PM2启动后端
```bash
cd /var/www/linmeng/linmeng-platform/server

# 创建PM2配置
cat > ecosystem.config.js << 'EOF'
module.exports = {
  apps: [{
    name: 'linmeng-api',
    script: 'src/app.js',
    instances: 2,
    exec_mode: 'cluster',
    env_production: {
      NODE_ENV: 'production',
      PORT: 3001
    }
  }]
}
EOF

# 启动服务
pm2 start ecosystem.config.js --env production

# 保存PM2配置
pm2 save

# 设置开机自启
pm2 startup
```

### 7.2 查看服务状态
```bash
pm2 status
pm2 logs linmeng-api
```

### 9.4 数据库备份
```bash
# 创建备份脚本
cat > /var/www/linmeng/backup.sh << 'EOF'
#!/bin/bash
BACKUP_DIR="/var/backups/mysql"
DATE=$(date +%Y%m%d_%H%M%S)
mkdir -p $BACKUP_DIR
mysqldump -u linmeng -p'your_password' linmeng > $BACKUP_DIR/linmeng_$DATE.sql
# 保留最近7天的备份
find $BACKUP_DIR -name "linmeng_*.sql" -mtime +7 -delete
EOF

chmod +x /var/www/linmeng/backup.sh

# 添加定时任务 (每天凌晨2点备份)
crontab -e
# 添加以下行
0 2 * * * /var/www/linmeng/backup.sh
```

## 十、安全加固

### 10.1 防火墙配置
```bash
# Ubuntu (ufw)
sudo ufw allow 22/tcp    # SSH
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
sudo ufw enable

# CentOS (firewalld)
sudo firewall-cmd --permanent --add-service=ssh
sudo firewall-cmd --permanent --add-service=http
sudo firewall-cmd --permanent --add-service=https
sudo firewall-cmd --reload
```

### 10.2 MySQL安全
```bash
# 只允许本地连接
sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf
# 确保 bind-address = 127.0.0.1

# 重启MySQL
sudo systemctl restart mysql
```

### 10.3 定期更新
```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 更新Node.js依赖的安全漏洞
npm audit fix
```

## 十一、常见问题

### Q1: 502 Bad Gateway
- 检查后端服务是否运行：`pm2 status`
- 检查端口是否正确：`netstat -tlnp | grep 3001`
- 查看错误日志：`pm2 logs linmeng-api`

### Q2: 数据库连接失败
- 检查MySQL服务：`sudo systemctl status mysql`
- 检查数据库配置：`.env.production`
- 检查用户权限：`mysql -u linmeng -p`

### Q3: SSL证书问题
- 检查证书状态：`sudo certbot certificates`
- 手动续期：`sudo certbot renew`
